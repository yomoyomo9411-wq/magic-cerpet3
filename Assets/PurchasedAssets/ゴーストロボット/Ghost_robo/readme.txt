オリジナル3Dモデル「ゴーストロボット」
製作者: RGBこーぼー

ふわふわ飛ぶロボットのお化けです。

Xなどで写真をアップする際にハッシュタグ（#三原色工房）
を付けていただけると作者が喜びます。

概要
VRChatでの使用を想定した3Dモデルです。
制作環境は「Maya」の為、他プログラムでの互換性は保証出来ません。

「Unity2022.3.22f1にて作成しております」

アップロードの際はVCC（VRChat Creator Companion）で作成したUnityProjectをご利用ください。

仕様
使用シェーダー: lilToon( https://lilxyzw.booth.pm/items/3087170 )

◇アバター情報
・パフォーマンスランク：Medium
・△ポリゴン数：9986
・マテリアルスロット数：2

◇ブレンドシェイプ

・各パーツの色替えギミック

◇内容物
・UnityPackage
・テスクチャPNGファイル
・テクスチャPSDファイル

・設定済みPrefabVariant付き

導入方法
Unity.2022.3.22f1（2022.3.6f1でも問題無し）と
VRChatCreatorCompanionをインストール後
Avatar用にCreateNewProjectを選択
Avatarを選択しliltoonシェーダー、ModularAvatarを
導入して起動してください。

起動後、購入したフォルダ内に
unityプロジェクトデータがありますので
そちらをクリック、インポート後
データ内のPrefabをHierarchyにドラッグアンドドロップし
上段メニューのVRChat SDK - Show Control Panel - Builder - Build & Publish for Windowsの順にクリックし、アバターをアップロードをすれば完了となります。
